# Hoist Maintenance Test

**Created by Albaraa Alghamdi**
HSE Engineer · 2026

A multilingual (English / Arabic / Hindi) safety knowledge test app for tower crane hoist maintenance workers.

## Features

- 10 questions based on OSHA, ASME B30.7, ISO 4309, EN 14492 standards
- 80% pass mark
- Three languages with full RTL support for Arabic
- Worker name and ID logged with every test
- Records saved permanently on each phone, no internet needed
- Supervisor PIN lock on records (default: 1234)
- Export records to CSV or PDF
- Installable to phone home screen, works offline

## How workers install on their phone

**Android (Chrome):** open the link → menu (⋮) → Install app

**iPhone (Safari):** open the link → share button → Add to Home Screen
